# Etape 4 - Preuves visuelles

Ce dossier central regroupe tous les visuels marketing demandes dans la checklist.

## Structure

- `captures/web/`: captures ecran des pages principales.
- `captures/resultats/`: resultat gratuit et rapport complet.
- `captures/pdf/`: PDF exemple.
- `visuels/`: visuels marketing statiques (SVG).
- `social/facebook/`: 3 images pretes a publier.
- `social/linkedin/`: 3 images pretes a publier.
- `email/`: 2 images pretes pour les courriels.

## Convention de nommage

Format conseille:

`YYYY-MM-DD_canal_contenu_format.png`

Exemples:

- `2026-04-04_web_home_desktop.png`
- `2026-04-04_facebook_resultat-gratuit_1200x630.png`
- `2026-04-04_email_rapport-complet_1200x900.png`

# Script de relance apres appel - Loi 25

## Relance J+1 (courriel)

Objet:
`Recap de notre appel Loi 25 + prochaine etape`

Modele:
`Bonjour [Prenom],`

`Merci encore pour notre echange.`

`Voici le recap:`
- `Priorite #1: [priorite]`
- `Priorite #2: [priorite]`
- `Prochaine action proposee: [Starter/Pro + action concrete]`

`Comme convenu, je vous envoie [proposition/document] pour vous aider a avancer sans vous disperser.`

`Lien utile: [lien rapport ou proposition]`

`Si vous etes d'accord, on valide la prochaine etape d'ici [date].`

`Au plaisir,`
`[Signature]`

## Relance J+4 (courriel court)

Objet:
`Petit suivi sur votre priorite Loi 25`

Modele:
`Bonjour [Prenom],`

`Petit suivi rapide: voulez-vous qu'on lance [Starter/Pro] cette semaine pour avancer sur [priorite principale]?`

`Je peux aussi vous proposer une version plus legere si vous voulez commencer en petit.`

`Bonne journee,`
`[Signature]`

## Relance finale (optionnelle, J+10)

Objet:
`On ferme le dossier ou on avance?`

Modele:
`Bonjour [Prenom],`

`Je vous ecris une derniere fois pour savoir si vous voulez qu'on avance sur votre plan Loi 25.`

`Si ce n'est pas le bon moment, aucun souci, je peux vous recontacter plus tard.`

`Merci,`
`[Signature]`

## Regles de relance
- Toujours garder un ton simple et respectueux.
- Une relance = une proposition concrete.
- Mettre a jour le pipeline apres chaque envoi.

# Loi 25 et site web : quoi verifier

Votre site web est souvent le premier point de collecte de donnees.
C'est donc un excellent point de depart pour une mise a niveau Loi 25 pragmatique.

## Checklist rapide
1. Formulaires
- Les champs sont-ils justifies?
- Le message de consentement est-il clair?

2. Politique de confidentialite
- Est-elle accessible facilement?
- Est-elle coherente avec ce que vous faites vraiment?

3. Cookies et traceurs
- Le texte est-il comprehensible?
- Le choix de l'utilisateur est-il respecte?

4. Courriels et notifications
- Les confirmations envoyees sont-elles claires?
- Les liens fonctionnent-ils correctement?

5. Acces et mots de passe
- Les comptes admin sont-ils limites au necessaire?
- Le MFA est-il active quand possible?

6. Outils relies
- CRM
- Outil courriel
- Formulaires tiers
- Analytics
Tous doivent suivre la meme logique de collecte et de protection.

7. Procedure d'incident
- Qui est responsable?
- Comment noter l'incident?
- Comment repondre rapidement?

## Erreurs frequentes
- Copier un texte legal generique sans lien avec la realite.
- Trop demander d'information dans les formulaires.
- Ne pas documenter les decisions prises.

## Approche recommandee
- Corriger d'abord les points visibles sur le site.
- Ajouter un mini processus interne.
- Reviser ensuite les outils plus complexes.

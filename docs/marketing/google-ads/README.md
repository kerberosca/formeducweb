# Etape 14 - Kit Google Ads

Ce dossier contient les elements prets a copier-coller pour lancer les campagnes Google Ads Loi 25.

## Fichiers
- `campagne-structure-loi25.md`: architecture campagnes et parametres.
- `mots-cles-loi25.csv`: base keywords par campagne/groupe.
- `rsa-assets-loi25.md`: 15 titres + 4 descriptions RSA.
- `extensions-google-ads-loi25.md`: sitelinks, accroches et extraits de site.
- `mots-cles-negatifs-loi25.md`: liste de negatifs de depart.
- `checklist-lancement-google-ads.md`: checklist execution dans Google Ads.
- `suivi-requetes-google-ads.csv`: journal requetes + decisions d'optimisation.

## Regle de travail
1. Copier les assets dans Google Ads.
2. Lancer petit (budget test).
3. Lire les requetes des 72 premieres heures.
4. Couper vite le trafic non qualifie.

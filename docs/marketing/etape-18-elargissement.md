# Etape 18 - Elargissement

## Objectif
Elargir l'offre seulement quand la machine d'entree est stable.

## Decisions prises
- Priorite absolue: conserver Loi 25 comme porte d'entree tant que la conversion est active.
- Conditions minimales avant elargissement:
  - 2 semaines consecutives avec signaux reels stables (Lead/Contact).
  - suivi pipeline applique sans trous.
  - messages principaux testes sur organique + paid.
- Ordre d'elargissement retenu:
  1. Renforcement des preuves (cas/exemples/resultats)
  2. Optimisation page service d'implantation
  3. Sequence de montee Starter/Pro
  4. Extension progressive vers services web / SEO / cybersecurite
